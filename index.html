Scopely by rushinbarot

Data mining Project

Install

run on a GNU/linux environment preferably compiled with GCC It also requires GDBM for on-disk hash, flex as a lexer, and the GNU standard C library All these are standard Linux libraries dmlex is the standalone lexer/tokenizer generate does all the counting with low level commands

Contact

(rushinbarot@gmail.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/rushinbarot/Scopely

get the source code on GitHub : rushinbarot/Scopely
